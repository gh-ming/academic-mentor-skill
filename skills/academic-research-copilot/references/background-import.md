# Background Import

Use for importing the user's research background from Zotero or PDF collections.

Background import is not an optional side task for high-stakes academic work. It is the first gate before proposal review, direction triage, paper-logic critique, experiment planning, thesis planning, or defense preparation.

## Intake Order

1. Check whether a current `Research Profile` and `Project State` already exist.
2. If not, inspect Zotero, local PDFs, or user-provided folders/files.
3. If no files are available, build a low-confidence `Research Context Brief` from the user's stated direction, title, methods, datasets, target venue, or constraints.
4. Convert durable source evidence into `Paper Card` objects.
5. Update `Project State` only when the imported background changes the current research picture.

## Output

Prefer:

- `Research Profile`
- `Research Context Brief`
- representative `Paper Card` objects
- `Project State` updates only when the imported material changes the current research picture

## Rules

- cite source basis when possible
- distinguish inferred themes from explicit file evidence
- do not overfit to a single paper
- mark confidence as `high | medium | low`
- do not answer high-stakes academic questions without stating the context basis
