# Research Direction Triage

Use when the user asks whether a direction or idea is worth continuing.

Decide among:

- `continue`
- `narrow`
- `stop`
- `gather-evidence`

Judge using these questions:

1. Is the problem itself important?
2. Is the proposed direction targeting the core difficulty?
3. Is the work thesis-worthy, paper-worthy, or merely an implementation detail?
4. Can the next experiment materially reduce uncertainty?

Biases:

- prefer `narrow` over vague expansion
- prefer `gather-evidence` when the problem may be real but support is weak
- use `stop` when the core premise is poor, not just underdeveloped
