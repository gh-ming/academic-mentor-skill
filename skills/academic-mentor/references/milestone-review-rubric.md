# Milestone Review Rubric

Use for stage reviews, quarterly planning, and go/no-go checks.

Judge:

1. Is the current work advancing the core problem?
2. What has actually been de-risked?
3. What uncertainty still blocks the next stage?
4. What is the next single action with highest information gain?

Prefer:

- smaller defensible milestones
- explicit go/no-go conditions
- evidence-producing tasks over activity accumulation
